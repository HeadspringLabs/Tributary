Buttons

Buttons can be easily styled by adding the .btn class to an element. Primary actions—most pages will have only one—can be signified by adding .btn-primary, which adds additional styling.

It's important to style as buttons only those elements that directly respond to user interactions. Styling visual elements as buttons causes confusion.
